## Sujet

Ubisoft se lance dans la création d'un nouveau jeu vidéo.
Dans cette optique, nous avons pour mission d'étudier les caractéristiques et les tendances du marché des jeux vidéos sur la plateforme de jeux en ligne Steam.

## Problématique

Le but : analyser un large jeu de données pour comprendre les facteurs qui influencent la popularité d'un jeu vidéo.
